# تبچی بدون دیلیت
---------------------------------
# کلون و نصب
```
git clone https://github.com/Telesud/TelesudTab
cd tabi
chmod +x tablighati
chmod +x tg
./tablighati install
```

# پیکربندی در هر یوزر فقط یکبار این دستورات رو بزنید
```
./tablighati config

```
